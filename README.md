# 📖 About

This portfolio is designed to showcase my journey as a **Software Engineer**, highlighting my technical skills, projects, experience, achievements, and competitive programming profile.

The primary goal of this portfolio is to provide recruiters, developers, and visitors with a fast, interactive, and visually appealing platform to learn more about my work.

Built using the latest version of **Next.js**, the application focuses on:

- Modern UI
- Responsive Design
- High Performance
- Clean Code
- Accessibility
- SEO Optimization

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/amansharma264/my-portfolio.git
```

---

## Navigate

```bash
cd my-portfolio
```

---

## Install Dependencies

Using npm

```bash
npm install
```

Using pnpm

```bash
pnpm install
```

Using yarn

```bash
yarn
```

---

## Start Development Server

```bash
npm run dev
```

Visit

```
http://localhost:3000
```

---

# 📦 Build for Production

```bash
npm run build
```

Run Production

```bash
npm start
```

---

# ⚙ Environment Variables

If required, create

```
.env.local
```

Example

```env
NEXT_PUBLIC_EMAIL=your@email.com
NEXT_PUBLIC_GITHUB=https://github.com/username
NEXT_PUBLIC_LINKEDIN=https://linkedin.com/in/username
```
