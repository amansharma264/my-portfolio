# 🚀 My Portfolio

A modern, fast, and animated personal portfolio website built with **Next.js**, **TypeScript**, and **Tailwind CSS**. It showcases my projects, GitHub activity, and blog posts — all wrapped in smooth animations and a clean design.

🔗 **Live Demo:** _(https://aman-sharma-portfolio-eso4.vercel.app/)_

---

## ✨ Features

- ⚡ **Next.js 16** with the App Router for fast, server-rendered pages
- 🎨 **Tailwind CSS 4** for a clean, responsive, utility-first design
- 🎬 **Framer Motion** for smooth, polished animations
- 📊 **GitHub Contribution Calendar** to display live coding activity
- ✍️ **MDX-powered Blog/Content** using `@next/mdx` and `gray-matter` for frontmatter parsing
- 🖍️ **Syntax-highlighted code blocks** via `rehype-pretty-code`
- 🧩 **Type-safe forms & data** with `zod`
- 🧑‍💻 Clean, reusable component architecture
- 📈 **Vercel Analytics** integrated out of the box
- 🌗 Responsive and optimized for all screen sizes

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ installed
- npm / yarn / pnpm / bun

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/amansharma264/my-portfolio.git
   cd my-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result 🎉
