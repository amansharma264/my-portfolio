# 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/amansharma264/my-portfolio.git
```

### Navigate to the Project

```bash
cd my-portfolio
```

### Install Dependencies

Using npm

```bash
npm install
```

Using pnpm

```bash
pnpm install
```

Using yarn

```bash
yarn
```

---

### Start the Development Server

```bash
npm run dev
```

Visit:

```
http://localhost:3000
```

---

# 📦 Build for Production

```bash
npm run build
```

Run the production server:

```bash
npm start
```
